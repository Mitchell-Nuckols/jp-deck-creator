Memrise/Anki Deck Generator

By エステティック (Aesthetic)

This page makes use of jisho.org to generate a .tsv (tab separated value) file that can be imported into Memrise or Anki. When creating a deck on Memrise, be sure to put the fields in this order: "Kanji", "Kana", "English", "Part of Speech" (make this section an attribute). The recommended configuration is to test on kanji, prompt with English, and show kana after test. Additionally, you can create a second level to test on kana and prompt with kanji to remember pronunciation. If you're using Memrise, just copy the text (not including the first row) and import it with the "Bulk Add Words" option under the "Advanced" menu.
